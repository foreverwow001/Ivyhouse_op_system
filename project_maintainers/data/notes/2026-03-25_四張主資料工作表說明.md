# 四張主資料工作表說明

更新日期：2026-03-31

本次新增四張工作表，作為銷售商品組成對照表之後的下一層資料整理基礎：

1. `2026-03-25_內包裝完成品主檔_template.csv`
2. `2026-03-25_生產_分裝_轉換扣帳規則表_template.csv`
3. `2026-03-25_單位換算規則表_template.csv`
4. `2026-03-25_外包裝材料主檔_template.csv`

## 2026-03-31 補充

- `2026-03-25_外包裝材料主檔_template.csv` 已擴充 `耗材分類` 欄位，沿用原檔名，但內容範圍已可同時維護 `外包裝耗材` 與 `內包裝耗材`。
- 外包裝主檔新增 `牛皮橫窗夾鏈袋(小)`、多款 `禮盒姓名貼`，以及 `膠捲`、`餅乾袋`、`蛋糕袋`、`椰棗小霧袋`、`糖果袋` 等內包裝耗材；其中 `禮盒姓名貼` 類別歸為 `出貨及行政耗材`，但仍屬正式入表、入扣帳項。
- 另新增 `2026-03-31_出貨及行政耗材總表_template.csv`，用於管理人工盤點且不納入系統正式扣帳的出貨與行政耗材。
- 另新增 `2026-03-31_內包裝耗材用量對照表_template.csv`，並補入 `適用包裝規格` 欄位，用於整理 `內包裝完成品` 對 `內包裝耗材` 的正式用量對照。

### 內包裝耗材扣帳 vs 轉換規則邊界

- `2026-03-31_內包裝耗材用量對照表_template.csv` 目前已覆蓋所有已確認的「內包裝完成品 -> 內包裝耗材」扣帳規則，包含：
	- 夏威夷豆塔、堅果塔、雪花餅、鳳梨酥的膠捲封膜
	- 奶油曲奇 / 西點餅乾 / 杏仁瓦片的單片袋裝
	- 瑪德蓮的單個袋裝
	- 椰棗與糖果的袋裝內包裝
- 下列 SKU 雖然目前未進入內包裝耗材用量對照表，但屬於已評估後刻意不納入：
	- `Q1-Q5` 奶油曲奇整包完成品、`G1-G5` 西點餅乾整包完成品、`C1-C7` 杏仁瓦片秤重完成品：這些主體的關鍵治理重點是 `拆包轉重量中介` 或 `秤重轉單片`，正式 owner 是 `2026-03-25_生產_分裝_轉換扣帳規則表_template.csv`；若其 sellable 包裝需扣 `PK0016 橫窗鋁箔夾鏈立袋`，仍由 `1-2026-03-24_銷售商品組成對照表_template.csv` 管理，不在內包裝耗材表重複建列。
	- `K1001-K4001` 濾掛咖啡單包：屬委外單包完成品，現階段不另建內包裝耗材扣帳規則。
- 因此，內包裝耗材表與轉換規則表的分工原則為：
	- `耗材扣帳`：用來定義做出某個內包裝完成品時，實際消耗哪些袋材、膜材等耗材。
	- `轉換規則`：用來定義完成品、重量中介、單片、分裝包之間的數量或單位轉換關係。

## 預填原則

- 已優先採用 `成品商品清單.xlsx` 的 `成品庫存` tab 作為現況主檔來源。
- 已引用 `1-2026-03-24_銷售商品組成對照表_template.csv` 的內容，把禮盒、組合、外包材、單片、單包與糖果 100g 內包裝等需求帶入。

- 對於目前檔案無法可靠推得的欄位，例如每包克重、每片克重、標準損耗率，先留空或標示 `待建立` / `待確認`。
