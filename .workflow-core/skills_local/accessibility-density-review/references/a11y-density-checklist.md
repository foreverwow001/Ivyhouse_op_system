# Accessibility And Density Checklist

## 基本檢查

- [ ] 主要文字對比足夠
- [ ] hover / focus / disabled 狀態清楚
- [ ] 重要按鈕與卡片可用鍵盤抵達
- [ ] 可點區域不會小到難以操作
- [ ] 表單 label、helper、error text 都清楚可見

## 營運系統密度檢查

- [ ] 區塊分群清楚，不靠大量細線硬切
- [ ] 文字層級清楚，主次資訊易掃描
- [ ] 卡片數量與資訊量平衡，不讓首頁變成資訊牆
- [ ] 品牌裝飾沒有壓過功能資訊
- [ ] 在較高密度場景下仍保有留白與視覺節奏
